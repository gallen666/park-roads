Part 3 · DTSI 评估方法论

DTSI 加权公式（专利核心）

DTSI = 0.257·政府 + 0.187·资产 + 0.143·运营
+ 0.120·企业 + 0.110·科研 + 0.093·员工 + 0.090·居民

权重来源：AHP 300 位行业专家评分
一致性比率 CR = 0.0283 < 0.1 ✓（学术显著）

7 主体维度详细评分

#主体权重5 项评估细分
1政府25.7%政策衔接 / 招商引资 / 监管效率 / 政绩 KPI / 公共服务
2资产持有方18.7%楼宇智能化 / 能源管理 / 资产估值 / 出租率 / 物业服务
3运营方14.3%物业数字化 / 后勤效率 / 安全管理 / 数据中台 / 应急响应
4入驻企业12.0%一站式服务 / 财税 / 法务 / 共享设施 / 业务赋能
5科研机构11.0%产学研对接 / 实验室共享 / 论文专利 / 中试基地 / 人才流转
6员工9.3%通勤 / 食堂 / 健身 / 培训 / 社交活动数字化
7居民/家属9.0%教育 / 医疗 / 商业 / 文化 / 社区数字化
总计35 个滑杆（7 主体 × 5 维度），每个 0-100 分

评分加权示例

某园区评估结果：
政府 65 × 25.7% = 16.7 | 资产 70 × 18.7% = 13.1
运营 55 × 14.3% = 7.9 | 企业 60 × 12.0% = 7.2
科研 50 × 11.0% = 5.5 | 员工 55 × 9.3% = 5.1
居民 45 × 9.0% = 4.1

DTSI 综合得分 = 59.6 → PDTMM L3 良好级
